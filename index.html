**CMSC422: Machine Learning** !!! Tip This is a live site for the ongoing Fall 2018 class Welcome to CMSC422: Machine Learning! This course broadly covers an introduction to machine learning and pattern recognition. Topics include: supervised learning, unsupervised learning, learning theory, deep learning, and reinforcement learning. This course will also cover applications of machine learning, including computer vision, natural language processing, bioinformatics, and autonomous navigation. # Announcements August 24, 2018: Welcome to CMSC422! - We look forward to meeting you on Monday 8/27 at 3:30pm!
